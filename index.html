The Operator

Mistromy. I design posters that look like recovered evidence, build 3D spaces that only exist for the render, and write software that keeps a heartbeat when I'm not looking.

Everything on this site is one person's output — the design, the code, the bad decisions and the fixes for them. The recurring theme across all of it is signal: machines waking up, transmissions half-received, systems insisting they're fine.

If a number on this site is live, it's really live. If it's not, it says so. That policy has occasionally been humbling.
